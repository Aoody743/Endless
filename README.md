# Endless 

(by Aoody)

Endless is expected to be my flagship project in the upcoming period

Useful to use in scenes including the following:

> Personal Blog
> 
> Knowledge & Document Base
> 
> Personal Showcase
> 
> Portfolio Showcase
> 
> Business Showcase., etc.

Expected to have the following features:

> A convenient UI designed by myself, using my design philosophy & standard.
> 
> An easy-to-use Markdown editor system
> 
> A alternative natural language editor system (May integrate AI composing feature)
> 
> An nice App used to edit the contents, availible at App Store & Android APK file
> 

This system is expected to be half-paid, that is, the App controling system would be paid but the web side of the project would be open-sourced.

I will start to develop the system after Jun 06th.
